LINE通知サービスにおける個人情報の取り扱い

取得する情報

本サービスのLINE公式アカウント（@704rqoiy）を友だち追加いただいた場合、LINEプラットフォームを通じて以下の情報を取得します。

  • LINE ユーザーID（LINEが発行する識別子。氏名・電話番号等は含まれません）
氏名・メールアドレス・電話番号・位置情報など、LINEユーザーID以外の個人情報は取得しません。

利用目的

取得したLINE ユーザーIDは、以下の目的のみに使用します。

  • スポラディックE層（Es）発生時のLINEプッシュ通知の送信

目的外の利用、第三者への提供、広告・マーケティング目的での使用は一切行いません。

保存場所・管理

保存場所Cloudflare Workers KV（米国 Cloudflare, Inc. のサーバー）
アクセス権限運営者（JQ3JQO）のみ
保存期間退会（LINEでのブロック）まで
暗号化Cloudflare の標準セキュリティに準拠

情報の削除方法

以下のいずれかの方法で、保存されたLINE ユーザーIDを削除できます。

  • LINEでブロック：公式アカウント（@704rqoiy）をブロックすると自動的に通知送信が停止されます。ただしKVからのデータ削除は翌営業日以降の手動対応となります。
  • メールで申請：yotsuzeki@gmail.com にLINE登録解除の旨をご連絡ください。速やかに削除します。

通知の確実性について

本サービスのLINE通知は以下の理由により送信を保証しません。

  • Cloudflare Worker の障害・メンテナンス時
  • LINE Messaging API のレート制限超過時（月間送信数の上限あり）
  • NICTデータの取得失敗・欠損時
  • ネットワーク障害時
重要な交信機会の判断は、モニター画面のリアルタイムデータを直接ご確認ください。LINE通知の遅延・未着に関して運営者は責任を負いません。
データ出典と利用条件

NICT（情報通信研究機構）データ

本サービスで表示するFxEs・MUFデータは、国立研究開発法人情報通信研究機構（NICT）の 電離圏観測ネットワークおよび短波通信予報業務から取得した観測値を使用しています。

利用形態非商用・個人利用・出典明記
加工の有無観測数値を読み取り、グラフ・カラーインジケーターとして表示
出典表記全ページに「NICT 電離圏観測ネットワーク」と明記
NICTポリシーhttps://www.nict.go.jp/policy/
本サービスはNICTの公式サービスではありません。NICTの観測データについてはNICT公式サイトを正とし、本サービスの表示はあくまで参考情報です。

NOAA SWPC（宇宙天気予報センター）データ

宇宙天気データ（X線・Kp指数・F10.7）は米国海洋大気庁（NOAA）宇宙天気予報センター（SWPC）の 公開データを使用しています。米国政府機関のデータはパブリックドメインです。
